# Codex Optimization Workspace

This directory is the reviewable mirror for the active Codex optimization setup.

- `context-map.md` distills recurring session patterns.
- `global-agents.md` mirrors the intended `~/.codex/AGENTS.md`.
- `skills/` mirrors user skills installed under `~/.agents/skills/`.
- `templates/` contains repo-local starter templates.
- `rule-audit.md` records advisory command-policy recommendations.

The mirror should be updated before installing changes into the active Codex
home, then committed in small milestones.
